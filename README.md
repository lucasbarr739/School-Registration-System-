# School Registration System

A JavaFX-based school registration system that simulates the process of managing students, courses, and enrollment through a graphical user interface. This project was developed as part of Rutgers University's Software Methodology course and demonstrates object-oriented design, event-driven programming, and desktop application development.

## Features

* Graphical user interface built with JavaFX
* Student registration and management
* Course enrollment and withdrawal
* Course section administration
* Schedule management
* Prerequisite validation
* Credit limit enforcement
* Time conflict detection
* Course roster generation
* Dynamic schedule viewing

## Technologies Used

* Java
* JavaFX
* Object-Oriented Programming (OOP)
* IntelliJ IDEA
* Git & GitHub

## Running the Application

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Configure JavaFX dependencies if necessary.
4. Run the main application class.
5. The application launches as a desktop GUI window.
